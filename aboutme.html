About Me

Overview

This was a university project in which I was tasked to recreate Light’s Out from Tiger Electronics. It was a one-week endeavor to implement all of the intricacies of this game.

This version of Light’s Out follows the rules of the original. It’s a single-player game in which each tile has two states, an On or Off state. The goal of the game is to set all tiles to their Off state. When a player selects a tile, it and its adjacent tiles get toggled to their opposite state. Each time the game restarts there is a randomized board.
